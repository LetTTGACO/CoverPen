## ⚡ Features
- 🚀 super fast and easy to use
- 🌈 6 different themes
- 🌠 100+ dev icons with option to upload custom icon

## 👩‍💻 Developing
This project was bootstrapped with [NextJs](https://nextjs.org/).



```shell
git clone https://github.com/LetTTGACO/CoverPen.git
cd CoverPen/site/
pnpm dev
```
